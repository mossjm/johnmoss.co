# Concept 08: Dark Mode

## Design Rationale

Dark background with light text. Sophisticated, modern, and easier on the eyes. Subtle color accents stand out against the dark canvas.

**Key Elements:**
- Near-black background (#0D0D0D)
- Off-white text (#E5E5E5)
- Subtle blue accent (#60A5FA)
- Space Grotesk font
- Generous spacing
- Minimal borders

**Target Feel:** Sophisticated, modern, premium. Like a well-designed developer portfolio.

**Best For:** Tech-forward audiences, investors familiar with modern startups.

## Run Locally

```bash
cd concept-08
npm install
npm run dev
```
