# Concept 04: Tech Executive

## Design Rationale

Modern, clean tech executive aesthetic. Dark text, subtle gray accents, sharp lines. Conveys competence in both agriculture and technology.

**Key Elements:**
- Pure white background
- Near-black text (#0A0A0A)
- Cool gray accents (#94A3B8)
- DM Sans font
- Bold section headers
- Subtle hover states

**Target Feel:** Competent, modern, tech-forward. Like a founder's personal page at a Series B company.

**Best For:** Tech investors, startup partners, modernization-focused customers.

## Run Locally

```bash
cd concept-04
npm install
npm run dev
```
