# Concept 05: Academic/Research

## Design Rationale

Scholarly, citation-like aesthetic. Inspired by academic papers and research publications. Conveys deep expertise and intellectual rigor.

**Key Elements:**
- Cream paper background (#FFFEF9)
- Crimson red accents (#8B2332) 
- Libre Baskerville for body text
- Numbered sections
- Footnote-style references
- Two-column layout for bio

**Target Feel:** Scholarly, authoritative, research-focused. Like a professor's personal page.

**Best For:** Research partners, academic collaborators, those who value expertise.

## Run Locally

```bash
cd concept-05
npm install
npm run dev
```
