# Concept 10: Ultra Minimal / Brutalist

## Design Rationale

Stripped to absolute essentials. No decoration, no color, just information. System fonts, raw HTML aesthetic, maximum clarity.

**Key Elements:**
- Pure white background
- Black text only (#000000)
- System font stack (no Google Fonts)
- No borders, no boxes
- Minimal spacing
- Text-only, no visual hierarchy tricks

**Target Feel:** Raw, honest, no-nonsense. Like a plain text file rendered in a browser.

**Best For:** Those who appreciate extreme simplicity and directness.

## Run Locally

```bash
cd concept-10
npm install
npm run dev
```
