# Concept 06: Single Page Scroll

## Design Rationale

Everything on one continuous scroll. No navigation clicking needed. Sections flow naturally with clear visual breaks. Perfect for mobile-first visitors.

**Key Elements:**
- Full-viewport hero section
- Clear section dividers
- Sticky nav dots on the side
- Smooth scroll behavior
- IBM Plex Sans font
- Alternating section backgrounds

**Target Feel:** Modern, seamless, app-like. Easy to scan on any device.

**Best For:** General audiences, those who want quick overview without clicking.

## Run Locally

```bash
cd concept-06
npm install
npm run dev
```
