# Concept 09: Two-Column Sidebar

## Design Rationale

Fixed sidebar navigation with content area. Classic web layout that works well for information-dense pages. Always visible navigation.

**Key Elements:**
- Fixed left sidebar (240px)
- White content area
- Light gray sidebar (#F8F9FA)
- Work Sans font
- Compact navigation
- Clear content sections

**Target Feel:** Professional, organized, easy to navigate. Like a well-structured documentation site.

**Best For:** Repeat visitors, those who want to jump between sections.

## Run Locally

```bash
cd concept-09
npm install
npm run dev
```
